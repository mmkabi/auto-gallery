# Three-Minute Demo Script

## 0:00-0:25 — Positioning

Show Auto Gallery, Lyra — AI Vehicle Concierge, and **Explore. Understand. Decide.** Read or summarize the competition disclosure. State clearly that Guided Demo is deterministic and Offline Assistance is rule-based.

## 0:25-1:15 — Guided Vehicle Journey

Start Guided Demo. Show the exterior reveal in fixed Onyx Black, validated driver-door pivot, and cabin entry. Pause while the cabin is visible, orbit around the dashboard and seats, then Resume.

Continue through steering, front-wheel coordination, wheel motion, and rock-suspension mode. Explain that suspension is a visual interaction demonstration, not physics.

## 1:15-1:45 — Truthful Asset Handling

Show cargo inspection. Explain that the unreliable hatch group is temporarily hidden instead of presenting a false opening animation.

Show the conceptual powertrain overlay. State that the source asset has no real engine mesh.

## 1:45-2:15 — Commerce Flow

Show the sample payment calculation and the labels **Demonstration Only** and **Not a Financing Offer**.

Show the local lead summary generated by the tour. It does not create a CRM record. Optionally demonstrate the manual consent form with fictional data, then show **Demo lead created locally** and clear it.

## 2:15-2:40 — Reliability

Use Exit, then Restart Guided Demo. Explain that every step has a timeout and recovery path, and each restart invalidates the previous run before resetting vehicle and UI state.

## 2:40-3:00 — Architecture and Close

State: "Live tool-calling was validated in isolated compatibility tests. The submitted competition build defaults to a deterministic Guided Demo because the third-party API gateway did not provide sufficiently reliable response latency."

Close with: "Auto Gallery turns vehicle discovery into a transparent, repeatable decision journey."
